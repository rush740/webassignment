Introduction to HTML by Mike Raleigh

Doctype

Doctype informs the browser about what type of document to expect

Example of the code

<!doctype html>

html

html is Hyper text markup language

Example of the code

 <html></html>

head

head is a container for metadata (data about data) placed between html tag and body tag

Example of the code

 <head></head>

body

The body tag defines the documents body

Example of the code

 <body></body>

title

The title tag defines the title of document

Example of the code

<title></title>

p

The p tag defines a paragraph

Example of the code

<p>This is a paragraph</p>

Working example

This is a paragraph

span

The span tag allows part of a text to be styled

Example of the code

<p>My mother has <span style="color:green">green</span> eyes.</p>

Working example

My mother has green eyes.

div

The div tag defines a division or a section in an HTML document

Example of the code

Working example

Lorem Ipsum
I am a div
dolor sit amet.

images & Alt

The HTML img tag is used to embed an image in a web page

Example of the code

 <img src="images/firefly.jpg" width="204" height="204" alt="firefly"/>

Working example

links

Links allow you to jump to another document

Example of the code

<a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

Working example

Visit W3Schools!

quote & blockquote

The q tag defines a short quotation

The blockquote tag defines a section by indenting text

Example of the code

 <q>to be or not to be</q>

Example of the code

 <blockquote cite="https://www.huxley.net/bnw/four.html"> <p>Words can be like X-rays, if you use them properly—they’ll go through anything. You read and you’re pierced.</p></blockquote>

Working example for quote

Famous shakespeare quote to be or not to be

Working example for blockuote

Words can be like X-rays, if you use them properly—they’ll go through anything. You read and you’re pierced.

abbreviations

The abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Example of the code

The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.

Working example

The WHO was founded in 1948.

meta

The meta tag defines metadata about an HTML document. Metadata is data (information) about data

Example of the code

<meta charset="UTF-8">

Working example
